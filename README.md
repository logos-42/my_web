# 绘画作品集

## 主要功能

1. **全屏查看**：点击图片可以全屏查看作品
2. **唯一凭证**：每张图片都有唯一ID凭证，类似NFT但不依赖区块链
3. **受欢迎度排序**：用户可以为喜欢的作品点赞，系统会记录并优先展示受欢迎的作品
4. **分页浏览**：支持大量图片的分页浏览
5. **多种排序方式**：按受欢迎度、随机排序、最新作品排序

## 使用方法

1. 双击 `start_gallery.bat` 启动本地服务器
2. 浏览器将自动打开画廊页面
3. 查看所有作品，可使用页面顶部的分页和排序控制

## 图片管理

### 本地图片

所有图片应放在 `finish` 文件夹中，命名格式为：`thumbnail_数字编号.jpg`

例如：
- thumbnail_1.jpg
- thumbnail_2.jpg
- thumbnail_3.jpg
- ...

## 详细图床使用指南

由于GitHub对仓库大小有限制，大量图片无法直接上传。以下是几种解决方案：

### 方案1：使用免费图床

推荐几个免费图床服务：

1. **ImgBB**（推荐）
   - 注册网址：https://imgbb.com/
   - 特点：免费账户无限上传，保存期限至少6个月
   - 上传方法：
     1. 注册并登录
     2. 批量上传图片（支持拖拽）
     3. 上传后获取直接链接（Direct Link）

2. **SM.MS**
   - 注册网址：https://sm.ms/
   - 特点：免费用户有5GB空间，简单易用
   - 上传方法：
     1. 注册账号
     2. 上传图片
     3. 复制图片链接

3. **Imgur**
   - 注册网址：https://imgur.com/
   - 特点：图片质量好，适合长期存储
   - 上传方法：
     1. 注册账号
     2. 创建相册
     3. 批量上传图片
     4. 获取直接链接

### 方案2：使用云存储服务

如果有大量图片且需要长期稳定存储，建议使用专业云存储：

1. **七牛云对象存储**（推荐）
   - 网址：https://www.qiniu.com/
   - 免费额度：10GB存储空间，每月10GB流量
   - 优势：国内访问速度快，有免费CDN加速
   - 设置教程：
     1. 注册账号并实名认证
     2. 创建存储空间（Bucket）
     3. 批量上传文件
     4. 设置访问权限为公开
     5. 获取域名和文件路径

2. **腾讯云COS**
   - 网址：https://cloud.tencent.com/product/cos
   - 新用户有一定免费额度
   - 优势：稳定性好，适合大量图片

3. **阿里云OSS**
   - 网址：https://www.aliyun.com/product/oss
   - 新用户试用期免费
   - 优势：企业级稳定性，易于管理

### 配置图床链接到画廊

上传图片到图床后，按照以下步骤配置到画廊：

1. 打开 `js/gallery-loader.js` 文件
2. 修改配置信息：

```javascript
// 配置对象
const config = {
  // 文件来源配置
  sourceType: 'remote',  // 修改为'remote'
  
  // 远程图床配置
  remoteCDN: {
      enabled: true,  // 修改为true
      baseUrl: 'https://你的图床网址/',  // 例如：https://i.imgur.com/
      fileList: [
          // 添加你上传的所有图片文件名
          'abc123.jpg',
          'def456.jpg',
          'ghi789.jpg',
          // ...更多图片
      ]
  },
}
```

3. 如果所有图片都有规律的命名（例如都是数字序列），你可以这样简化配置：

```javascript
// 示例：批量生成图片链接
let fileList = [];
for(let i = 1; i <= 200; i++) {
    fileList.push(`image_${i}.jpg`);
}
config.remoteCDN.fileList = fileList;
```

## "想要"按钮功能

画廊中的每张图片都有"想要"按钮，功能如下：

1. 点击"想要"按钮，表示你喜欢这件作品
2. 系统会记录每件作品被"想要"的次数
3. 默认情况下，作品会按受欢迎度（被"想要"的次数）排序
4. "想要"数据会保存在浏览器本地存储中
5. 你可以在页面顶部切换不同的排序方式：
   - **按受欢迎度**：将被"想要"次数最多的作品排在前面
   - **随机排序**：每次刷新页面都会随机排序，但受欢迎的作品有更高概率出现在前面
   - **最新作品**：假设编号越大的作品越新，按编号倒序排列

## 自定义配置

### 修改排序权重

```javascript
// 排序配置
sortMode: 'popular',  // 默认排序模式: 'popular', 'random', 'newest'
popularityWeight: 3,  // 受欢迎度权重系数（影响随机排序时的权重计算）
```

增加 `popularityWeight` 值可以使受欢迎作品在随机排序中出现概率更高。

### 修改样式

画廊的样式定义在 `css/gallery.css` 文件中，可以根据需要修改。

## 故障排除

- 如果端口被占用，程序会自动寻找可用端口
- 如果无法启动，请检查Python是否正确安装
- 如果图片不显示，检查`finish`文件夹是否存在以及图片命名是否正确
- 如果样式有问题，尝试清除浏览器缓存
- 如果"想要"按钮不工作，检查浏览器是否允许本地存储
