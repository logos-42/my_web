# 我的个人网站

欢迎来到我的个人网站项目！这是一个展示我的能力和项目的博客类网站。

## 网站结构

- **index.html** - 网站主页，展示最新动态
- **blog.html** - 博客页面，展示我的项目和学习心得
- **about.html** - 关于我页面，介绍我的个人信息
- **styles.css** - 网站样式文件

## 如何使用

1. 双击 `index.html` 文件即可在浏览器中打开网站
2. 通过顶部导航栏可以在不同页面间切换
3. 博客页面会定期更新我的最新项目和学习心得

## 功能特点

- 简洁明了的页面布局
- 响应式设计，适配手机和电脑
- 易于维护和扩展的代码结构
